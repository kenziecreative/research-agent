Create both maps following the structure from 02-CONTEXT.md. Read each type template's credibility hierarchy.

**curriculum-research.md:**
- Read `.claude/reference/templates/types/curriculum-research.md` for credibility hierarchy and phase pattern
- Curriculum research has a distinctive phase pattern: domain landscape, practitioner reality, skill decomposition, misconceptions, best practice, field trajectory, competing approaches, existing programs
- Domain landscape/subject matter → academic primary (peer-reviewed research highest); web-search secondary
- Practitioner reality → social-signals primary (practitioner communities, forums, conference talks are high credibility for ground truth); web-search secondary
- Skill decomposition/misconceptions → academic + social-signals primary (need both textbook and practitioner perspectives)
- Best practice/field trajectory → academic primary; web-search + social-signals secondary
- Competing approaches/existing programs → web-search primary; social-signals secondary (blog posts, tutorials from credible practitioners)
- Government data (labor statistics, occupational frameworks) relevant → regulatory secondary for workforce context phases
- Domain-specific sources: Educational repositories, professional credentialing databases (from domain-specific playbook educational resources hook), standards bodies

**person-research.md:**
- Read `.claude/reference/templates/types/person-research.md` for credibility hierarchy and phase pattern
- Person research covers: professional background, publications, public statements, reputation, connections
- Professional background/career → web-search primary; social-signals (LinkedIn) secondary; regulatory secondary (for executives — SEC filings list officers)
- Publications/IP → academic primary (published papers); domain-specific (patent search by inventor) secondary
- Public statements/media → web-search primary; social-signals (social media, forums) secondary
- Reputation/influence → social-signals primary; web-search secondary
- Financial connections (if executive/founder) → financial + regulatory primary
- Domain-specific sources: Patent search (by inventor), professional registries

Both maps: YAML frontmatter with active-channels, phase-grouped sections, primary/secondary, rationale lines, domain-specific sources section, no synthesis phases. Web-search explicit where applicable.
